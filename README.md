# A1-1 프롬프트 관리 프로그램

컴퓨터에게 명령하는 말(파이썬)을 처음 배우며 만든 프로젝트입니다.
자주 쓰는 AI 프롬프트를 저장하고 관리하는 CLI 프로그램입니다.

## 프로젝트 소개

AI를 사용하다 보면 좋은 프롬프트를 매번 다시 찾게 됩니다.
이 프로그램은 프롬프트를 카테고리별로 저장하고,
즐겨찾기와 검색으로 빠르게 찾을 수 있도록 만들었습니다.

## 개발 환경

- Python 3
- Git / GitHub

## 실행 방법

1. 저장소를 클론합니다.

   git clone (저장소 주소)

2. 프로젝트 폴더로 이동합니다.

   cd A1-1

3. 프로그램을 실행합니다.

   python3 main.py

## 기능 목록

| 번호 | 기능 | 설명 |
|:---:|:---|:---|
| 1 | 전체 목록 보기 | 저장된 모든 프롬프트 출력 |
| 2 | 프롬프트 추가 | 제목/내용/카테고리 입력해 저장 |
| 3 | 프롬프트 검색 | 키워드로 프롬프트 찾기 |
| 4 | 즐겨찾기 목록 | 즐겨찾기한 프롬프트만 보기 |
| 5 | 카테고리별 보기 | 번호 선택으로 카테고리 조회 |
| 6 | 상세 보기 | 프롬프트 전체 내용 확인 |
| 7 | 즐겨찾기 토글 | 즐겨찾기 추가/해제 |
| 8 | 종료 | 프로그램 종료 |

## 사용 예시

프로그램을 실행하면 메뉴가 나타납니다.
원하는 기능의 번호를 입력해 사용합니다.
