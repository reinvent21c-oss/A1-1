# A1-1 프롬프트 관리 프로그램

컴퓨터에게 명령하는 말(파이썬) 처음 배우고 작업 이력 남기기.
자주 쓰는 AI 프롬프트를 저장하고 관리하는 CLI 프로그램입니다.

## 개발 환경

- Python 3
- Git / GitHub

## 실행 방법

1. 저장소를 클론합니다.
2. 아래 명령어로 실행합니다.

   python3 main.py

## 기능 목록

1. 전체 목록 보기
2. 프롬프트 추가
3. 프롬프트 검색
4. 즐겨찾기 목록 보기
5. 카테고리별 보기
6. 프롬프트 상세 보기
7. 즐겨찾기 추가/해제 (토글)
8. 종료
